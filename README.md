# Exercices d'entretien technique — Ada Tech School

Ce dépôt héberge les **énoncés** des exercices utilisés lors des entretiens techniques de simulation d'Ada Tech School (coaching Natlar Labs).

- Un dossier par candidat·e sous `sujets/`.
- Les énoncés sont publiés **juste avant** chaque entretien.
- Les **solutions ne sont pas publiées ici**.

## Consignes candidat·e (usage de Git)

1. Clone ce dépôt : `git clone https://github.com/natlarlabs/adatechschool.git`
2. Ouvre ton dossier `sujets/<ton-prenom-nom>/` et lis le `README.md`.
3. Crée une branche à ton nom : `git checkout -b prenom-nom`
4. Résous les exercices, en **committant au fur et à mesure** avec des messages clairs.
5. Pousse ta branche : `git push origin prenom-nom`
6. Ouvre une **Pull Request**.

> La façon dont tu utilises Git (branches, commits, messages, PR) fait partie de ce qui est observé pendant l'entretien. Pas de stress : pense à voix haute, on regarde ta démarche.
